Features

Raid Frame Support

Works with ANY raid frame addon: ElvUI, Grid, Grid2, VuhDo, HealBot, Blizzard CompactRaidFrames, Shadowed Unit Frames, PitBull4, Lime, and more.

Nameplate Support

Uses Ascension's backported C_NamePlate API with scheduled timer for continuous aura updates.

Arena Frames

Display important CC on arena enemy frames for competitive PvP awareness.

Party Frames

Track debuffs on party members with configurable icon sizes.

Priority System

Intelligent priority-based aura display shows the most important debuff first.

LibGetFrame Integration

Dynamic frame detection works with any unit frame addon automatically.
