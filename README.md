> **Displays CC/debuff information on unit frames as icons** - WoW 3.3.5a (Project Ascension)

## Description

Displays important crowd control and debuff information as large icons on various unit frames. Works with arena, party, raid, focus, target, and nameplates.

## Features

* **Raid Frame Support** - Works with ANY raid frame addon (ElvUI, Grid, Grid2, VuhDo, HealBot, SUF, etc.)
* **Nameplate Support** - Uses Ascension's backported C_NamePlate API
* Arena, Party, Focus, and Target frame support
* Priority-based aura display
* Configurable icon sizes and positions
* ENG and RUS locale support

## New in v6.9 - Ascension Raid/Nameplate Support

* Added support for raid units (raid1-raid40)
* Integrated LibGetFrame-1.0 for dynamic raid frame detection
* Added nameplate support using Ascension's C_NamePlate API
* New "Raid Frames" and "Nameplates" tabs in Options UI
* Scheduled timer for continuous aura updates on nameplates
